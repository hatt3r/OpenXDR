# OpenXDR (Phase 1: Ready or Not)

OpenXDR is a lightweight, Go-based experimental Extended Detection and Response (XDR) system designed for learning cybersecurity telemetry pipelines, endpoint monitoring, and distributed event collection.

This project is built from scratch using Go and gRPC, with a MacBook acting as the central server and a Windows machine acting as the endpoint agent.
